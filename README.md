Text Summarization Tool

A Streamlit-based Natural Language Processing (NLP) web app that automatically summarizes long articles or text passages into concise summaries using basic text processing techniques.

This tool is simple, interactive, and visually appealing ‚Äî designed to help users quickly extract key information from lengthy content.

*Features:

- Text Summarization using tokenization, word frequency, and sentence scoring

- Dynamic Animated Background with gradient effects for a modern UI

- Stylish Interface with colored headings, custom fonts, and formatted widgets

- Summary Statistics including sentence count, word count, and compression ratio

- Sample Articles provided for quick testing

- Instant Summarization without heavy ML models
